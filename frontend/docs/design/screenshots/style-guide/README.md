# Capturas del Style Guide

Añadir aquí las capturas de pantalla de la página Style Guide (`http://localhost:4200/style-guide`).

## Archivos requeridos

| Archivo | Descripción | Qué capturar |
|---------|-------------|--------------|
| `typography-colors.png` | Tipografía y Colores | Secciones "Typography" y "Colors" del Style Guide |
| `buttons-alerts.png` | Botones y Alertas | Secciones "Buttons" y "Alerts" mostrando todas las variantes |
| `forms.png` | Formularios | Secciones de Form Input, Form Textarea, Form Select |
| `cards.png` | Cards | Service Cards, Stats Cards, Order Cards |
| `modal-auth.png` | Modal y Auth Form | Componentes Modal y Auth Form |
| `layout.png` | Layout Components | Header (variantes), Footer, Sidebar |

## Instrucciones

1. Ejecutar `npm run start` en el frontend
2. Navegar a `http://localhost:4200/style-guide`
3. Capturar cada sección usando las herramientas del navegador o una extensión de screenshots
4. Guardar en esta carpeta con los nombres indicados
5. Resolución recomendada: 1440px de ancho

## Alternativa con Docker

```bash
docker compose -f docker-compose.dev.yml up frontend
# Navegar a http://localhost:4200/style-guide
```

# Capturas de Figma

Añadir aquí las capturas del diseño en Figma demostrando los 5 principios de comunicación visual.

## Archivos requeridos

| Archivo | Descripción | Qué capturar en Figma |
|---------|-------------|----------------------|
| `jerarquia-visual.png` | Jerarquía Visual | Pantalla mostrando la escala tipográfica (títulos grandes a texto pequeño) |
| `contraste.png` | Contraste | Pantalla con colores semánticos (success verde, error rojo, warning amarillo) |
| `alineacion.png` | Alineación | Pantalla mostrando el grid de 8px y alineación de elementos |
| `proximidad.png` | Proximidad | Formulario o sección mostrando agrupación de elementos relacionados |
| `repeticion.png` | Repetición | Pantalla con cards repetidas (Service Cards, Stats Cards) |

## Instrucciones

1. Abrir el archivo de Figma del proyecto
2. Seleccionar la pantalla/componente que mejor demuestre cada principio
3. Exportar como PNG a esta carpeta con el nombre indicado
4. Resolución recomendada: 1440px de ancho (diseño de referencia)
